### Dependency(종속성)

Linux, 특히 Fedora 시스템의 경우, OS의 기정의 인스톨시에, Maya가 동작하기 위해 필요한 시스템 라이브러리의 일부가 인스톨 되지 않습니다. yum 를 사용하면, 추가가 필요한 시스템 의존 라이브러리를 검색해, 필요한 런타임 라이브러리를 구성할 수 있습니다.
